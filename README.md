# AI Travel Guide - Интелигентен Туристически Асистент

AI Travel Guide е модерно уеб приложение, което осигурява на потребителите готов план за екскурзия, изготвен от вградения в приложението ИИ. Проектът съчетава текстова обработка и компютърно зрение, за да предложи план за посещение на забележителности и препоръки за хотели.

## Основни функции

- Изготвяне на туристически план: Потребителят задава дестинация по която да му бъде генериран план.
- Разпознаване на дестинация от снимка: Потребителят може да качи снимка на забележителност, а вграденият ИИ модел автоматично идентифицира локацията.
- Персонализирано планиране: Генериране на детайлен план за престой от избрани от потребителя брой дни.
- Интерактивен избор на хотели: Филтриране на предложения за настаняване според желания рейтинг (звезди).
- Мултимодална комуникация: Използване на модел Gemini 2.0 Flash, който обработва едновременно текст и изображения.
- Адаптивен дизайн: Напълно оптимизиран интерфейс за работа на десктоп и мобилни устройства (Responsive Design).

## Технологичен стек

- Backend: [FastAPI](https://fastapi.tiangolo.com/) (Python)
- Engine: [Google Gemini API](https://ai.google.dev/) (gemini-2.0-flash)
- Frontend: HTML5, CSS3 (Custom Variables & Media Queries), JavaScript (Vanilla)
- Инструменти: Uvicorn, Python-multipart, Dotenv

## Използвани библиотеки 

Проектът използва следните ключови библиотеки, описани в `requirements.txt`:

* FastAPI: Основен уеб фреймуърк за изграждане на API ендпоинтите.
* google-genai: Официалната библиотека на Google за интеграция с модела Gemini 2.0 Flash. Поддържа мултимодални заявки (текст + изображения).
* uvicorn: ASGI сървър за стартиране на FastAPI приложението с висока производителност.
* python-dotenv: Използва се за сигурно зареждане на API ключовете от `.env` файл, избягвайки тяхното публикуване в публични хранилища.
* python-multipart: Необходима за FastAPI, за да може сървърът да обработва качването на файлове (снимки) през HTML формите.
* Pydantic: За валидация на данните и управление на настройките чрез Python типове.
